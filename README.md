# 周金霞男朋友养成计划 V3.1 Cloud ❤️

这是一个专为情侣设计的互动网页应用，包含甜蜜积分、愿望清单、恋爱商店、**温馨相册**、**姨妈助手**等功能，并支持云端数据同步。

## ✨ 主要功能

### V3.1 新增功能 🎉
1.  **历史记录管理**：支持删除误操作的历史记录（分数保持不变），让记录更整洁。
2.  **姨妈助手**：贴心记录生理期，自动推算下次日期，做最懂她的男朋友。
3.  **每日情话**：每次打开都有不同的甜言蜜语，给生活加点糖。

### 核心功能
1.  **甜蜜积分系统**：
    *   记录每一次心动瞬间，加分/减分都有仪式感。
    *   **求饶模式**：减分时触发趣味求饶弹窗，增加互动乐趣。
    *   **等级称号**：从“新手男友”到“家庭帝位”，见证感情升温。
2.  **愿望清单 (Wishlist)**：
    *   一起写下想做的事（看电影、旅游、吃大餐）。
    *   打卡完成，通过进度条直观展示恋爱成就。
3.  **温馨相册 (Photo Album)**：
    *   **拍立得风格**：记录生活中的每一个美好瞬间，每张照片都带有日期和心情。
    *   **智能压缩**：支持上传本地图片（自动压缩以节省流量）或粘贴网络图片链接。
    *   **大图浏览**：点击照片即可查看高清大图。
4.  **恋爱商店**：
    *   使用积分兑换奖励（如按摩券、免做家务卡、清空购物车等）。
5.  **云端同步**：
    *   基于 LeanCloud，支持多设备数据实时同步，换手机也不怕数据丢失。
6.  **唯美 UI 设计**：
    *   动态流光背景、樱花飘落特效、毛玻璃质感。
    *   完美适配手机端操作。

## 📱 如何安装到手机？

### 方法一：添加到主屏幕（最推荐，永久免费）🌟
这种方法不需要复杂的打包，不需要证书，体验和原生 App 几乎一样（全屏运行，无地址栏）。

1.  将 `index.html` 文件上传到任意免费静态网站托管（如 GitHub Pages, Vercel, 或自己的服务器）。
2.  在苹果手机 Safari 浏览器中打开该链接。
3.  点击底部中间的 **分享按钮**（带有向上箭头的方框）。
4.  下滑找到并点击 **“添加到主屏幕”**。
5.  点击右上角的 **“添加”**。
6.  桌面就会出现一个爱心图标的 App，点击即可全屏运行！

### 方法二：打包成安装包 (IPA) 🛠️
如果你坚持想要一个 `.ipa` 文件通过爱思助手安装，由于苹果系统的封闭性，步骤会比较复杂，且通常需要**付费购买证书**。

**准备工作：**
*   电脑安装 **HBuilderX** (uni-app 官方开发工具)。
*   准备苹果开发者证书（Certificate）和描述文件（Provisioning Profile）。（通常需要苹果开发者账号 $99/年，或者在淘宝/第三方平台购买个人签名证书）。

**打包步骤：**
1.  打开 HBuilderX，选择“文件” -> “新建” -> “项目”。
2.  选择 “5+App” -> “默认模板”，输入项目名称。
3.  将本项目中的 `index.html` 内容复制并覆盖新建项目中的 `index.html`。
4.  打开 `manifest.json`，配置 App 图标、启动图等信息。
5.  点击顶部菜单 “发行” -> “原生App-云打包”。
6.  选择 “iOS (ipa包)”，填入你的证书信息（证书私钥密码、profile文件等）。
7.  点击“打包”，等待完成后下载 IPA 文件。
8.  使用爱思助手连接手机，将 IPA 文件导入安装。

## ⚠️ 注意事项
*   本项目使用了 LeanCloud 进行数据存储，请确保在 `index.html` 或设置页面中正确配置了你的 AppID 和 AppKey。
*   iOS 系统对隐私权限要求严格，如果遇到无法通过爱思助手安装的情况，通常是证书失效或未信任导致。
*   **相册说明**：为了保证同步速度，本地上传的图片会自动进行压缩，且建议相册存储照片数量控制在 20 张以内。
